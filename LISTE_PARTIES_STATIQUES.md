## Recommandations

Pour rendre l'application plus dynamique, considérer :
1. Créer une table `cms_content` pour stocker les textes statiques
2. Créer une interface d'administration pour modifier ces contenus
3. Remplacer les arrays statiques par des appels API/database
4. Externaliser les configurations dans des fichiers de configuration
